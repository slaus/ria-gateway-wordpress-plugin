== Description ==

The plugin helps you accept payments using Ria Money Transfers.

== Installation ==

1. Upload 'woocommerce-ria-money-gateway' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access 'admin.php?page=wc-settings&tab=checkout&section=ria' in order to change the description and the details provided in the email
